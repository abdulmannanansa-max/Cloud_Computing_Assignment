Technologies Learned So Far

This page outlines the key technologies and concepts covered in the Cloud Computing course.

☁ Google Cloud Platform (GCP)

  • Google Cloud Storage (GCS) – object storage & static site hosting
  • IAM & Service Accounts – managing permissions and access
  • GCP Console and Cloud SDK (gcloud CLI)

📄 GitHub & Git

  • Version control with Git (add, commit, push)
  • GitHub Repositories – storing and managing source code
  • GitHub Secrets – securely storing credentials
  • GitHub Actions – automating CI/CD workflows

⚙ CI/CD Concepts

  • Continuous Integration – automatic testing & building on push
  • Continuous Deployment – automatic release to production
  • YAML workflow syntax for GitHub Actions pipelines

🌐 Static Website Hosting

  • HTML, CSS & JavaScript for static websites
  • Configuring GCS bucket for public website hosting
  • Main page and error page configuration
  • Making bucket objects publicly accessible

🔒 Security & IAM

  • Service Account JSON key generation
  • Assigning roles (Storage Admin / Object Creator)
  • Storing secrets safely in GitHub Secrets

🔨 Cloud Infrastructure

  • Cloud storage concepts: buckets, objects, regions
  • Storage classes (Standard, Nearline, Coldline, Archive)
  • Cloud deployment and hosting fundamentals
